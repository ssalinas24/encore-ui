This component provides styles and a demo for the the Angular-UI Bootstrap Tabs plugin, which is included as a dependency for EncoreUI. Usage is the exact same as demoed on the Angular-UI Bootstrap documentation.

Disclaimer

  • Only the default horizontal tabs are supported by these styles.
  • vertical, pills, and justified tabs are currently unsupported.

DO

  • Do use tabs for complex forms that use option tables. Tabs are a good way to display "either/or" choices to the user.
  • Do use tabs to show closely related information cleanly.

DON'T

  • Don't use non-standard tab patterns, like the accordion example below.
  • Don't use more than 5 tabs. If more than 5 tabs are required, then consider whether tabs are the right pattern.
  • Don't use tabs with a total width of more than 500px. If the width is more than 500px, consider shorter tab labels.
  • Don't use tabs as navigation with entire pages inside. Rather than putting extensive or lengthy content in a tabbed area, create a new page with an additional link in the left hand nav.
